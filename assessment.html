Questão 1

1) Quando é recomendando a criação de um site separado para mobile? Seria este o caso deste site da GK Fashion ou do site integrado de Interfaces com JS - SPA?

É recomendado quando a maior parte dos acessos são feitos por dispositívos móveis e quando há orçamento/necessidade suficiente a versão mobile separada da interface. No caso da GK Fashion há uma vitrine de roupas que provavelmente será visualizada por uma proporção maior parte do publico mobile se comparado ao single Page pela atividade de olhar produtos (shopping) ser mesmo considerado uma forma de lazer.

Questão 2

2) No caso de um CSS específico para print, como ele deve ser inserido?

Ele deve ser inserido dentro de uma folha de estilos css linkada ao documento html, através da tag @media. Exemplo: @media print{ /**regras css para impressao*/ }

Questão 3

3) O elemento audio permite como elemento‐filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento audio e quando devem ser utilizados?

A tag source permite definir atributos de midia como 'media' que define a forma da visibilidade no navegador 'type' para definição do formato da midia bem como seus codecs.
Na maioria dos casos não é desejado que o audio seja reproduzido instantamente, nas exceções á esses casos pode-se definir o atributo 'autoplay' no elemento 'audio' para definir que a musica seja reproduzida assim que o elemento audio for carregado, e o atributo 'muted' para reproduzir a midia mutada, também é possível reproduzir a midia novamente quando ela chegar ao fim através do atributo 'loop' bem como habilitar botões para controle de pausa/play,volume etc dentro do elemento a definir o atributo 'controls'

Questão 4

4) O elemento video permite como elemento‐filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento video e quando devem ser utilizados?

Os atributos permitidos pelo source são 'src' definindo o caminho do video, seja em arquivo ou url, 'type' definindo o formato da midia e os codecs tal como é utilizada quando definida dentro de uma tag '< audio >'. O elemento video herda todos os atributos explicados acima do elemento audio: 'loop','muted','controls' e 'autoplay' e torna possível o redimensionamento do video através das tags 'width' -- que define a largura do video -- e 'height' -- que define sua altura, esses valores entretanto não podem ser porcentagem

Questão 5

5) No caso do canvas, HTML 5 nunca, em momento algum, fez qualquer animação. Isso porquê o HTML 5 é uma simples linguagem de marcação. Dessa forma, o que é necessário para criar animações com HTML5? O que são APIs do HTML5?

Para criar animações com o HTML5 é necessário a manipulação do elemento HTML 'Canvas' pelo javascript através de coordenadas que devem ser desenhadas e formas pré-definidas como a função 'fillRect()' que recebe como parametros as coordenadas do ponto inicial e tamanho do retangulo.
APIs do HTML5 são serviços como funções comumente utilizadas que foram integradas nativamente ao HTML5 um exemplo disso foi o suporte introduzido à reprodução de video e audio pelas tags respectivas detalhadas nas questões anteriores, previamente á essa época era necessário um plugin à parte para reprodução de videos nos navegadores como Adobe Flash ou Quicktime, que muitas vezes não era compatível à todas as plataformas(navegadores).
